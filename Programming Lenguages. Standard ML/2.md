### Example: Reverse

```sml
fun reverse(x,z) = if null(x) then z
                else reverse(tl(x), hd(x)::z);

reverse([2,3,4], [1]) =
reverse([3,4], 2::[1]) = reverse([3,4], [2, 1]) =
reverse([4], 3::[2, 1]) = reverse([4], [3, 2, 1]) =
reverse([], 4::[3, 2, 1]) = reverse([], [4, 3, 2, 1]) =
[4, 3, 2, 1]
```

Reverse is **O(n)** where **n** is lenght of **x**
